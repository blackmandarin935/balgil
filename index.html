교과 연계 — 이 앱이 실제로 쓰는 수학

1. 조도의 거리 역제곱 법칙 (이차함수)
가로등에서 멀어질수록 밝기는 거리의 제곱에 반비례해 줄어듭니다. 여러 가로등의 빛은 더해집니다.

E(p) = Σ k / (d² + r₀²) · d = 가로등까지 거리, r₀ = 보정 상수

2. 두 점 사이 거리 (제곱근과 실수)

d = √( (x₂−x₁)² + (y₂−y₁)² ) → Math.sqrt(dx*dx + dy*dy)

3. 안전 점수와 경로 비용 (일차식·최적화)
각 구간의 안전 점수를 0~1로 매기고, 어두울수록 "멀게 느껴지는" 가상 거리를 씌운 뒤 최단 경로를 찾습니다.

score = w₁·조도 + w₂·CCTV + 0.10 − 0.30·신고수 비용 = 실제거리 × ( 1 + α·(1 − score) )

α(안전 민감도)를 0으로 두면 최단 경로와 같아지고, 크게 올릴수록 돌아가더라도 밝은 길을 고릅니다. 슬라이더를 움직이며 확인해 보세요.
